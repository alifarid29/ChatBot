# 📚 Bookshop Chatbot

A Python-based intelligent assistant designed to help users interact with a bookshop system. This project uses Natural Language Processing (NLP) to understand user intents and provide helpful responses through a dedicated interface.

## 🚀 Features
* **Intent Classification:** Uses a Deep Learning model to categorize user questions (e.g., greetings, book availability, hours).
* **NLP Pipeline:** Implements tokenization and lemmatization using **NLTK**.
* **Graphical User Interface:** A clean, easy-to-use UI built with **Tkinter**.
* **Model Training:** Includes a Jupyter Notebook showing the training process on a custom dataset.

## 📂 Project Structure
* `bookshop_chatbot.ipynb`: The training pipeline where the model is built.
* `bookshop_chatbot_model.h5`: The pre-trained Keras model.
* `chatbot_ui.py`: The main application script to run the chatbot.
* `intents (1).json`: The knowledge base containing patterns and responses.
* `words.pkl` & `classes.pkl`: Serialized data used for text processing.
